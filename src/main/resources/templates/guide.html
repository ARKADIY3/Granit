Руководство пользователя

1. Вход в систему

Регистрация нового пользователя Гость

1
Нажмите кнопку "Регистрация" в верхнем меню
2
Заполните все обязательные поля формы (имя, email, пароль)
3
Нажмите кнопку "Зарегистрироваться"
4
После успешной регистрации вы будете автоматически перенаправлены на страницу входа

Авторизация Гость

1
Нажмите кнопку "Войти" в верхнем меню
2
Введите ваш email и пароль
3
Нажмите кнопку "Войти"

2. Работа с заказами

Создание нового заказа Админ

1
В главном меню выберите "Заказы" → "Добавить заказ"
2
Заполните данные о клиенте (ФИО)
3
Выберите камень из списка доступных
4
Выберите услугу из списка доступных
5
Укажите вес камня и стоимость заказа
6
Нажмите кнопку "Сохранить"

Редактирование заказа Админ

1
В списке заказов нажмите "Редактировать" у нужного заказа
2
Внесите необходимые изменения в данные заказа
3
Нажмите кнопку "Сохранить" для применения изменений

3. Управление камнями

Добавление нового камня Админ

1
В главном меню выберите "Камни" → "Добавить камень"
2
Выберите название камня из выпадающего списка
3
Укажите технические характеристики (твердость по Моосу, плотность)
4
Заполните цвет и тип камня
5
Нажмите кнопку "Сохранить"

4. Управление услугами

Добавление услуги Админ

1
В главном меню выберите "Услуги" → "Добавить услугу"
2
Выберите тип услуги из выпадающего списка
3
Нажмите кнопку "Сохранить"

5. Управление пользователями

Поиск пользователей Админ

1
В главном меню выберите "Пользователи"
2
Введите ФИО пользователя в поле поиска
3
Нажмите кнопку "Поиск"

Удаление пользователя Админ

1
В списке пользователей найдите нужного пользователя
2
Нажмите кнопку "Удалить" напротив выбранного пользователя
3
Подтвердите удаление
